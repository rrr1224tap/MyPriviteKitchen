# 微信点餐小程序

本项目使用微信原生小程序和腾讯 CloudBase 云开发。

当前阶段为 MVP 第一阶段，先搭建小程序基础目录和页面骨架，后续再按任务拆解逐步开发登录、菜单、购物车、订单和商家接单流程。

## 当前目录

```text
miniprogram/
  app.js
  app.json
  app.wxss
  pages/
    common/launch/
    user/
    merchant/
  components/
  utils/
```

## 打开方式

使用微信开发者工具打开本项目根目录，然后确认 `miniprogram/app.json` 中注册的页面路径没有报错。
